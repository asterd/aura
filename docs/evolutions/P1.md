## Gap 1 — SharePoint Graph API live fetcher

### Contesto

Il file `aura/adapters/connectors/sharepoint.py` implementa correttamente:
- Normalizzazione ACL (allow/deny user+group) via `NormalizedACL`
- Risoluzione group ID da DB
- Cursor incrementale basato su `modified_at`

**Manca**: il metodo `_fetch_documents` lancia `NotImplementedError` se `credentials.extra["documents"]` non è presente. Non c'è integrazione con Microsoft Graph API.

### Approccio architetturale deciso

Usare **ACL-push al momento dell'ingestione** (non delegated token per ogni query):

1. Le credenziali di servizio (client credentials OAuth2) scansionano SharePoint via Graph API
2. Per ogni documento, si recuperano i permessi via `GET /drives/{id}/items/{item_id}/permissions`
3. I permessi vengono normalizzati come lista di `"user:email@domain.com"` (email lowercase)
4. Questi vengono salvati in Qdrant nel payload del chunk: `acl_allow_users: ["alice@contoso.com"]`
5. Al retrieve, il filtro Qdrant usa `UserIdentity.email.lower()` per matchare `acl_allow_users`

**Identificatore utente**: si usa l'**email** come chiave universale, non l'`okta_sub`. L'email è disponibile sia in SharePoint Graph (campo `user.email` / `userPrincipalName`) sia nel JWT Okta (claim `email`). Questo elimina la dipendenza da gruppi Okta sincronizzati.

**Fix critico applicato** in `filter_builder.py`: il filtro usava `identity.okta_sub` ma l'ACL SharePoint contiene email → allineato a `identity.email.lower()`.

La freschezza degli ACL dipende dalla frequenza del job di sync (raccomandato: ogni 15 minuti tramite ARQ cron job).

### Implementazione: `GraphApiClient`

Creare file: `aura/adapters/connectors/graph_client.py`

```python
from __future__ import annotations

import time
from dataclasses import dataclass, field
from typing import Any
import httpx

GRAPH_BASE = "https://graph.microsoft.com/v1.0"
TOKEN_URL = "https://login.microsoftonline.com/{tenant_id}/oauth2/v2.0/token"


@dataclass
class _TokenCache:
    access_token: str = ""
    expires_at: float = 0.0


class GraphApiClient:
    """Thin async client per Microsoft Graph API.

    Gestisce OAuth2 client credentials con cache del token in-memory.
    Thread-safe per uso in contesto asyncio single-process.
    """

    def __init__(
        self,
        *,
        tenant_id: str,
        client_id: str,
        client_secret: str,
        http_client: httpx.AsyncClient | None = None,
    ) -> None:
        self._tenant_id = tenant_id
        self._client_id = client_id
        self._client_secret = client_secret
        self._http = http_client or httpx.AsyncClient(timeout=30.0)
        self._cache = _TokenCache()

    async def get_token(self) -> str:
        """Restituisce access token valido, rinnovandolo se scaduto (con 60s buffer)."""
        if self._cache.access_token and time.monotonic() < self._cache.expires_at - 60:
            return self._cache.access_token

        url = TOKEN_URL.format(tenant_id=self._tenant_id)
        resp = await self._http.post(
            url,
            data={
                "grant_type": "client_credentials",
                "client_id": self._client_id,
                "client_secret": self._client_secret,
                "scope": "https://graph.microsoft.com/.default",
            },
        )
        resp.raise_for_status()
        data = resp.json()
        self._cache.access_token = data["access_token"]
        self._cache.expires_at = time.monotonic() + int(data.get("expires_in", 3600))
        return self._cache.access_token

    async def get(self, path: str, params: dict[str, str] | None = None) -> dict[str, Any]:
        """GET a Graph API path, gestendo 429 rate limit con retry."""
        token = await self.get_token()
        headers = {"Authorization": f"Bearer {token}"}
        resp = await self._http.get(f"{GRAPH_BASE}{path}", headers=headers, params=params)
        if resp.status_code == 429:
            retry_after = int(resp.headers.get("Retry-After", "10"))
            import asyncio
            await asyncio.sleep(retry_after)
            token = await self.get_token()
            headers = {"Authorization": f"Bearer {token}"}
            resp = await self._http.get(f"{GRAPH_BASE}{path}", headers=headers, params=params)
        resp.raise_for_status()
        return resp.json()

    async def paginate(self, path: str, params: dict[str, str] | None = None):
        """Itera su tutte le pagine di una risposta Graph paginata (via @odata.nextLink)."""
        data = await self.get(path, params)
        while True:
            for item in data.get("value", []):
                yield item
            next_link = data.get("@odata.nextLink")
            if not next_link:
                break
            token = await self.get_token()
            resp = await self._http.get(next_link, headers={"Authorization": f"Bearer {token}"})
            resp.raise_for_status()
            data = resp.json()

    async def aclose(self) -> None:
        await self._http.aclose()
```

### Implementazione: `SharePointGraphFetcher`

Creare file: `aura/adapters/connectors/sharepoint_graph.py`

```python
from __future__ import annotations

import logging
from collections.abc import AsyncIterator
from datetime import UTC, datetime
from typing import Any
from uuid import UUID

from aura.adapters.connectors.graph_client import GraphApiClient
from aura.domain.contracts import ResolvedCredentials

logger = logging.getLogger(__name__)

# Tipi MIME supportati per indicizzazione testo
INDEXABLE_MIME_TYPES = {
    "application/pdf",
    "application/vnd.openxmlformats-officedocument.wordprocessingml.document",
    "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet",
    "application/vnd.openxmlformats-officedocument.presentationml.presentation",
    "text/plain",
    "text/html",
    "text/markdown",
    "application/json",
}

# Dimensione massima file in bytes (default 50 MB)
MAX_FILE_SIZE_BYTES = 50 * 1024 * 1024


class SharePointGraphFetcher:
    """
    Fetcher che usa Graph API per scansionare SharePoint Drive.

    Restituisce documenti nel formato atteso da SharePointConnector.load_documents:
    dict con chiavi: external_id, title, source_path, source_url, content_type,
    modified_at, raw_text (o raw_bytes_ref), acl, is_deleted.

    ACL: recupera i permission di ogni item via Graph /permissions endpoint
    e li normalizza nel formato {allow: ["user:...", "group:..."], deny: []}.
    """

    def __init__(self, *, graph_client: GraphApiClient) -> None:
        self._graph = graph_client

    async def fetch(
        self,
        credentials: ResolvedCredentials,
        cursor: str | None,
    ) -> AsyncIterator[dict[str, Any]]:
        """Entry point principale. Restituisce async iterator di raw documents."""
        return self._fetch_impl(credentials, cursor)

    async def _fetch_impl(
        self,
        credentials: ResolvedCredentials,
        cursor: str | None,
    ) -> AsyncIterator[dict[str, Any]]:
        extra = credentials.extra or {}
        site_url: str = str(extra["site_url"])
        drive_id: str | None = extra.get("drive_id")  # type: ignore[assignment]
        folder_path: str = str(extra.get("folder_path", "/"))

        site_id = await self._resolve_site_id(site_url)
        if drive_id is None:
            drive_id = await self._resolve_default_drive(site_id)

        base_path = f"/sites/{site_id}/drives/{drive_id}"
        if folder_path and folder_path != "/":
            item_path = f"{base_path}/root:{folder_path}:/children"
        else:
            item_path = f"{base_path}/root/children"

        async for raw_doc in self._walk_items(
            base_path=base_path,
            items_path=item_path,
            cursor=cursor,
        ):
            yield raw_doc

    async def _walk_items(
        self,
        *,
        base_path: str,
        items_path: str,
        cursor: str | None,
    ) -> AsyncIterator[dict[str, Any]]:
        """Walk ricorsivo di tutti gli item in un Drive folder."""
        async for item in self._graph.paginate(items_path, params={"$top": "200"}):
            if "folder" in item:
                # Ricorsione su subfolders
                child_path = f"{base_path}/items/{item['id']}/children"
                async for child in self._walk_items(
                    base_path=base_path,
                    items_path=child_path,
                    cursor=cursor,
                ):
                    yield child
                continue

            if "file" not in item:
                continue

            mime = item.get("file", {}).get("mimeType", "")
            if mime not in INDEXABLE_MIME_TYPES:
                logger.debug("Skipping non-indexable file: %s (%s)", item.get("name"), mime)
                continue

            size = item.get("size", 0)
            if size > MAX_FILE_SIZE_BYTES:
                logger.warning("Skipping oversized file: %s (%d bytes)", item.get("name"), size)
                continue

            # Filtro incrementale: salta se modified_at <= cursor
            modified_str = item.get("lastModifiedDateTime", "")
            if cursor and modified_str and modified_str <= cursor:
                continue

            # Recupera permessi
            item_id = item["id"]
            acl = await self._fetch_acl(base_path, item_id)

            # Scarica contenuto testo
            raw_text = await self._download_text(base_path, item_id, mime)

            yield {
                "external_id": item_id,
                "title": item.get("name", ""),
                "source_path": item.get("parentReference", {}).get("path", "") + "/" + item.get("name", ""),
                "source_url": item.get("webUrl"),
                "content_type": mime,
                "modified_at": modified_str,
                "raw_text": raw_text,
                "acl": acl,
                "acl_required": True,
                "is_deleted": False,
            }

    async def _resolve_site_id(self, site_url: str) -> str:
        """Risolve site ID da URL SharePoint."""
        from urllib.parse import urlparse
        parsed = urlparse(site_url)
        hostname = parsed.hostname
        path = parsed.path.rstrip("/")
        data = await self._graph.get(f"/sites/{hostname}:{path}")
        return data["id"]

    async def _resolve_default_drive(self, site_id: str) -> str:
        """Recupera il drive principale del sito."""
        data = await self._graph.get(f"/sites/{site_id}/drive")
        return data["id"]

    async def _fetch_acl(self, base_path: str, item_id: str) -> dict[str, list[str]]:
        """
        Recupera permessi di un item e li normalizza.

        Formato output: {"allow": ["user:<email_or_id>", "group:<displayName>"], "deny": [], "inherited": true}
        """
        try:
            data = await self._graph.get(f"{base_path}/items/{item_id}/permissions")
        except Exception as exc:
            logger.warning("Failed to fetch ACL for item %s: %s", item_id, exc)
            return {"allow": [], "deny": [], "inherited": True}

        allow: list[str] = []
        inherited = True

        for perm in data.get("value", []):
            # Salta link anonimi
            if perm.get("link", {}).get("scope") == "anonymous":
                continue

            if not perm.get("inheritedFrom"):
                inherited = False

            roles = perm.get("roles", [])
            if not any(r in roles for r in ("read", "write", "owner")):
                continue

            granted_to = perm.get("grantedTo") or {}
            granted_to_identities = perm.get("grantedToIdentities") or []

            # Singolo user/group
            if granted_to:
                user = granted_to.get("user", {})
                group = granted_to.get("group", {})
                if user.get("email"):
                    allow.append(f"user:{user['email']}")
                elif user.get("id"):
                    allow.append(f"user:{user['id']}")
                if group.get("displayName"):
                    allow.append(f"group:{group['displayName']}")

            # Multiple identities (sharing link con persone specifiche)
            for identity in granted_to_identities:
                user = identity.get("user", {})
                group = identity.get("group", {})
                if user.get("email"):
                    allow.append(f"user:{user['email']}")
                if group.get("displayName"):
                    allow.append(f"group:{group['displayName']}")

        return {"allow": allow, "deny": [], "inherited": inherited}

    async def _download_text(self, base_path: str, item_id: str, mime: str) -> str | None:
        """
        Scarica contenuto file. Per ora restituisce None — il testo viene
        estratto da LlamaIndex durante l'ingestione tramite raw_bytes_ref.

        Implementazione futura: per file di testo puro, scaricare direttamente.
        """
        # LlamaIndex gestisce il parsing — restituiamo il download URL come ref
        # Il ConnectorSyncService deve gestire raw_bytes_ref per i file binari
        return None
```

```python
    self,
    credentials: ResolvedCredentials,
    cursor: str | None,

    # Priorità 1: fetcher iniettato (per test e ambienti mock)
```

### Contesto

```typescript
  spaces: Space[];
  selectedIds: string[];
  onChange: (ids: string[]) => void;

        </svg>
      </button>

      {open && (
        <>
          {/* Overlay per chiudere */}
          <div className="fixed inset-0 z-20" onClick={() => setOpen(false)} />
          <div
            className="absolute bottom-full mb-2 left-0 w-72 rounded-xl shadow-xl overflow-hidden z-30"
            style={{ backgroundColor: "var(--surface-raised)", border: "1px solid var(--border)" }}
          >
            <div className="px-3 py-2" style={{ borderBottom: "1px solid var(--border)" }}>
              <p className="text-xs font-semibold" style={{ color: "var(--muted-foreground)" }}>
                KNOWLEDGE SPACES
              </p>
            </div>

            {/* Free Chat option */}
            <button
              onMouseDown={(e) => { e.preventDefault(); onChange([]); setOpen(false); }}
              className="w-full flex items-center gap-2 px-3 py-2 text-left hover:opacity-80"
              style={{ borderBottom: "1px solid var(--border)" }}
            >
              <span
                className="w-4 h-4 rounded flex-shrink-0 flex items-center justify-center"
                style={{
                  border: `2px solid ${selectedIds.length === 0 ? "var(--accent)" : "var(--border)"}`,
                  backgroundColor: selectedIds.length === 0 ? "var(--accent)" : "transparent",
                }}
              >
                {selectedIds.length === 0 && (
                  <svg className="w-2.5 h-2.5" fill="none" stroke="var(--accent-foreground)" viewBox="0 0 24 24">
                    <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={3} d="M5 13l4 4L19 7" />
                  </svg>
                )}
              </span>
              <span className="text-sm" style={{ color: "var(--foreground)" }}>Free Chat</span>
              <span className="text-xs ml-auto" style={{ color: "var(--muted-foreground)" }}>No RAG</span>
            </button>

            {spaces.length === 0 && (
              <div className="px-3 py-4 text-xs text-center" style={{ color: "var(--muted-foreground)" }}>
                No spaces available
              </div>
            )}

            {spaces.map((space) => {
              const selected = selectedIds.includes(space.space_id);
              return (
                <button
                  key={space.space_id}
                  onMouseDown={(e) => { e.preventDefault(); toggle(space.space_id); }}
                  className="w-full flex items-center gap-2 px-3 py-2 text-left hover:opacity-80"
                  style={{ borderBottom: "1px solid var(--border)" }}
                >
                  <span
                    className="w-4 h-4 rounded flex-shrink-0 flex items-center justify-center"
                    style={{
                      border: `2px solid ${selected ? "var(--accent)" : "var(--border)"}`,
                      backgroundColor: selected ? "var(--accent)" : "transparent",
                    }}
                  >
                    {selected && (
                      <svg className="w-2.5 h-2.5" fill="none" stroke="var(--accent-foreground)" viewBox="0 0 24 24">
                        <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={3} d="M5 13l4 4L19 7" />
                      </svg>
                    )}
                  </span>
                  <div className="flex flex-col items-start min-w-0">
                    <span className="text-sm truncate w-full" style={{ color: "var(--foreground)" }}>
                      {space.name}
                    </span>
                    {space.description && (
                      <span className="text-xs truncate w-full" style={{ color: "var(--muted-foreground)" }}>
                        {space.description}
                      </span>
                    )}
                  </div>
                </button>
              );
            })}

            <div className="px-3 py-2 text-xs" style={{ color: "var(--muted-foreground)" }}>
              Tip: usa # nel messaggio per menzionare uno space
            </div>
          </div>
        </>
      )}
    </div>
  );
}
```

### Comportamento atteso

1. Nuova chat → SpaceSelector mostra "Free Chat" selezionato
2. Utente clicca → dropdown mostra tutti gli spaces disponibili con checkbox
3. Selezione multipla possibile (RAG su più knowledge bases)
4. "Free Chat" deseleziona tutti gli spaces
5. Il badge cambia colore se almeno uno space è attivo (accent color)
6. Durante lo streaming il selector è disabled
